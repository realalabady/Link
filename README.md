# Link Bloom

Welcome to Link Bloom! This is a modern, full-stack-ready React app built with Vite, TypeScript, shadcn-ui, and Tailwind CSS.

## Getting Started

Clone the repository and install dependencies:

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
```

Start the development server:

```sh
npm run dev
```

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deployment

You can deploy this app to any modern hosting provider (Vercel, Netlify, etc.).

## Custom Domains

Configure your hosting provider to use a custom domain as needed.
